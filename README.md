### Summary

IndexedDB + Dexie.js + Vue CLI , cms edit , display app

***
### Setup

* npm install -g @vue/cli

* vue create my-project

* cd my-project

* start server

***
### npm install

* dexie

npm install --save dexie

* marked

npm install -save marked


*  moment

npm install --save moment

***
### start server

npm run serve

* build project, deploy

npm run build
